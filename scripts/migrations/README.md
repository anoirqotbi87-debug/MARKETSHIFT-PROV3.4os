# Migrations Directory

Contains major rewrites and migrations for the application.

## Files

- `rewrite_app.cjs` - Application structure rewrite
- `rewrite_dashboard.cjs` - Dashboard component rewrite
- `rewrite2.cjs` - Secondary rewrite pass

## Apply

```bash
npm run migrate
```

⚠️ **WARNING**: Migrations may cause breaking changes. Test thoroughly before applying to production.
